# LED_PI4J_JAVA

# Instructions

###### Install Java
###### git clone https://github.com/Pi4J/pi4j-example-minimal
###### cd pi4j-example-minimal/
###### mvn clean package
###### cd target/distribution
###### ls -l
###### sudo ./run.sh 
###### This project was building using [PI4J](https://pi4j.com/getting-started/minimal-example-application/)
